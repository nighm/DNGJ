# 资产管理系统自动化测试开发计划

## 一、项目概述

### 1.1 项目目标
- 实现资产管理系统的自动化测试框架
- 开发完整的功能测试用例集
- 建立性能测试方案
- 生成详细的测试报告

### 1.2 技术栈选择
- 编程语言：Java（已安装 OpenJDK 21.0.6）
- 自动化测试框架：Selenium
- 浏览器：Microsoft Edge（已安装 - v134.0.3124.95）
- 性能测试工具：JMeter
- 构建工具：Maven
- 测试框架：TestNG
- 报告框架：Allure
- IDE：Cursor（已具备）

### 1.3 强制要求
- 在进行任何操作前，必须先检查是否已经存在相关配置或实现
- 严格使用 Edge 浏览器进行测试，不得使用其他浏览器
- 所有代码修改必须经过测试验证
- 保持配置文件和代码的一致性
- 遵循已有的项目结构和命名规范
- 所有测试用例必须包含详细的注释和文档
- 每个功能模块必须有对应的测试用例
- 测试数据必须从配置文件或数据文件中读取
- 所有测试结果必须记录到日志文件中
- 测试失败时必须自动截图保存
- 定期备份测试数据和配置文件
